# START

Project đã kết nối workflow core tại `{{WORKFLOW_ROOT}}`.

## Bắt đầu nhanh

Gõ:

```text
/start
```

Kỳ vọng:
1. Bootstrap wizard chạy theo dạng hỏi-đáp từng câu.
2. Ngôn ngữ hỏi-đáp: tiếng Việt.
3. Dùng bridge config tại `.workflow/config.yaml`.
4. Sau bootstrap, gợi ý `/status` hoặc `/plan <ten-ke-hoach>`.

## Lệnh khác

- `/status`
- `/plan <ten-ke-hoach>`
- `/review`
- `/test plan`
- `/test run`
- `/deploy`
