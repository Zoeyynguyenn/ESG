1. GPU 자원  사용  관련  ( 동적  할당 , 멀티  노드  사용
등 )
Q. 하나의  세션 ( 컨테이너 ) 에  제한  시간이  있을까요 ? 예를  들어 , 사용하지  않는
세션에  대해서  계속해서  유지가  되는지  궁금합니다 .
A. 사용자분께서  세션 ( 컨테이너 ) 을  생성하시게  되면  저희  관리  서버  쪽에서  GPU/CPU Util, 네
트워크  I/O 정보  등을  일정  간격으로  실시간  세션 ( 컨테이너 ) 데이터를  수집하고  있습니다 .
자원  회수  정책의  경우 , 해당  정보들을  바탕으로  특정  기준에  미치지  못할  시  해당  세션 ( 컨테이
너 ) 을  회수 ( 삭제 ) 하는  정책입니다 . ( 단 , 해당  정책은  동적  할당  정책을  사용할  때에만  적용되며 ,
추론 (Serving) 기능  이용  시에는  적용되지  않습니다 .)
본  서비스에서  동적  할당을  선택하는  경우 , 수집된  raw 데이터  기반으로  최근  n 시간  동안  GPU 의
Cuda Utilization 평균값이  1% 미만인  경우 , 해당  세션 ( 컨테이너 ) 를  삭제하고  자원을  회수하고
있습니다 .
자원  회수  정책으로  인해  세션이  종료되는  케이스는  세션  로그에  "idle-utilization" 로  인해  세션
이  종료된  것을  확인할  수  있습니다 .
CPU Utilization 도  고려될  수  있으며 , AI 학습을  위해  꾸준히  GPU 를  돌리신다면  확보하신
